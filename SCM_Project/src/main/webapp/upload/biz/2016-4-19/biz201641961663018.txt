다음 조건에 맞는 폼을 작성한 후 유효성 검사를 자바스크립트 작성하세요.
0. 회원가입 클릭시 빈 곳은 입력하도록 메시지 출력되고 포커스 되도록 합니다.

1. ID 
        size=10, name=id, id=id

2. ID중복검사 
        클릭시 중복 체크하기 이벤트 실행하기(함수명:idcheck)

3. 비밀번호 
        name=pass, id=pass

4. 주민번호  
       앞자리 : size=6, maxlength=6, name=jumin1, id=jumin1           
                숫자로 6자리 입력 후 뒷자리로 포커스 이동하기(함수명:move)
                숫자가 아닌 경우에는 숫자만 입력하라고 메시지 출력하기
       뒷자리 : size=7, maxlength=7, name=jumin2, id=jumin2

5. E-Mail
     앞부분 :  name=email, id=email
     뒷부분 :  name=domain, id=domain
               select 태그에서 선택시 뒷부분에 선택된 값 나타나고 뒷부분이 readOnly되게 한다.
             
               

6. 성별 
       name=gender, id=gender1(남자), name=gender, id=gender2(여자)
       두 개중 한개는 반드시 선택하도록 합니다.

7. 취미 : name=hobby id=hobby1(취미), name=hobby id=hobby2(게임), name=hobby id=hobby3(운동)
          name=hobby id=hobby4(등산), name=hobby id=hobby5(낚시)
          취미는 2개이상 선택하도록 합니다.

8. 우편번호
      앞자리 : size=3 maxlength=3 name=post1 id=post1
      뒤자리 : size=3 maxlength=3 name=post2 id=post2

9. 주소 
      size=50 name=address id=address

10. 자기소개
     rows=10 cols=50 name=intro id=intro
