조명 : SC Management (Successful Career Management)
 
각 게시판에 들어갈 기능 : 등록, 조회, 검색, 삭제, 수정 + 댓글 

1. 역할분담

나석균
- 비지니스 게시판
 
정지호
- 회원가입 및 로그인
- 메시지 기능

이가연
- SNS 게시판

황희천
- 질문 게시판


2. 개요

 가. 개발자 SNS

 나. 구직자와 IT회사 간에 연결

 다. 개발자들간에 네트워크 제공
 
 라. 자신의 포트폴리오 사이트가 되며 많은 사람들과 공유하고 정보를 얻을 수 있다.

 마. 구직, 이직 등이 쉬워지며 채용할 회사에서는 구체적으로 원하는 인재를 찾아 채용할 수 있다.

 

3. 구현기능

 가. 게시판 별 기능

  - 공통 기능

   1) 글 쓰기 (파일 or 사진 업로드)

   2) 내 글 수정

   3) 내 글 삭제

   4) 글 검색

  - SNS 게시판

   1) 좋아요 기능

   2) 댓글

  - 비지니스 게시판

   1) 이력서 지원 기능 (개인프로필 연동)

   2) 관심 업체 등록

  - message 게시판

   1) 사용자 간 메시지 송수신

   2) 메시지 정렬 기능

  - 질문 게시판

   1) 질문에 대한 답변 작성 (댓글 or 답변글)


4. 설계의 주안점

 가. 계정 생성

 나. 다양한 검색

 다. 추천기능 (사람 and 회사)

 라. 이력서 지원
 


5. 사용기술 및 개발환경

 사용 기술 : java, oracle, jsp, jquery, jstl, css3, html5, mybatis, spring

 개발 환경 : 이클립스, STS, MAVEN
